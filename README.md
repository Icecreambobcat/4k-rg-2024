# 4k-rg-2024
A 4k rhythm game built in python for school

## Dependancies
- [Python 3.12](https://www.python.org/downloads/)
- [Pygame 2.1.0](https://www.pygame.org/wiki/GettingStarted)

### Python dependancies
- pathlib
  - `pip3 install pathlib`
- argparse
  - `pip3 install argparse`
